# Agent rules

## Math formatting

When writing mathematics, never use LaTeX or math delimiters (`$...$`, `$$...$$`, `\(...\)`, `\[...\]`, `\boxed{}`, `\gcd`, etc.). Zed does not render math, so these show up as literal dollar signs and backslashes.

Write all expressions in plain Unicode/ASCII instead — for example: `6a + 10b + 15c`, `x²`, `√n`, `≤`, `≥`, `×`, `·`, `⁻¹`, `aₙ`, `gcd(6, 10, 15) = 1`. Keep variables and equations inline as readable text.
